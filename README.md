# BankProject

## Description
This project is a full Bank Management System, including front-end and back-end components. It handles clients, accounts, transactions, and other banking operations.

## Features
- Client management (add, edit, delete)
- Account management
- Transaction management (deposit, withdraw, transfer)
- Reporting and summaries

## How to Run
1. Open the solution in Visual Studio.
2. Make sure the database connection string is set correctly.
3. Build and run the application.
4. Use the UI to interact with the system.

## Technologies Used
- C# .NET Framework (Windows Forms)
- SQL Server
